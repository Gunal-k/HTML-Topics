Case Insensitive Example

This is a case insensitive example. HTML tags are not case sensitive.

For example, <H1> and <H1> are treated the same.

However, it is recommended to use lowercase for consistency and readability.

HTML attributes are also case insensitive, but it's best practice to use lowercase.

For example, <IMG SRC="image.png" ALT="Image"> is the same as <IMG SRC="image.png" ALT="Image">

Always use lowercase for HTML tags and attributes to avoid confusion.
